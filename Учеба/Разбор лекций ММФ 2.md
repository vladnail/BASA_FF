## 1.1 Асимптотическое разложение:

 Ряд $\displaystyle \sum_{n=0}^{+\infty} c_{n}(z-z_{0})^n$ называется асимптотическим разложением функции $f(z)$ при $z\to z_{0}$, если:
$$\lim_{ z \to z_{0} } \frac{f(z)- \displaystyle \sum_{n=0}^{N}c_{n}(z-z_{0})^n}{(z-z_{0})^N}=0$$
 $$f(z)- \sum_{n=0}^{N}c_{n}(z-z_{0})^n=O((z-z_{0})^{N+1})$$

Обычная сходимость: $\displaystyle\lim_{ N \to +\infty }\left( f(z)- \sum_{n=0}^{N}c_{n}(z-z_{0})^n \right)=0$
***Свойство 1:*** Если разложение $\exists$, то оно единственное. 

***Свойство 2:*** Разложение разных функцией могут совпадать. 

***Свойство 3:*** Разложение можно складывать, умножать, интегрировать (и дифференцировать, но с осторожностью).

 , где $A(x)$ - называется амплитудой, а $s(x)$ - фазой.

$$s(t) \approx s_{0} + s_{0} '' \frac{(t-t_{0})^2}{2}$$

## 1.3 Метод стационарной фазы:


$$F(\lambda)=\int_{a}^{b}A(t)e^{i \lambda s(t)} dt ; \, A(t), \,s(t) \in \mathbb{R}$$

а) ${} \forall t \in [a,b] \, s'(t) \neq 0: {}$

![[Pasted image 20260209010557.png|400]]
- Основной вклад от краев; 
- Полупериоды не компенсируются;
$\implies$ по частям.

$$F(\lambda)= \int_{a}^{b} \frac{A(t)}{i \lambda s'(t)} d(e^{i \lambda s(t)})= \frac{Ae^{i \lambda s}}{i \lambda s'} \bigg| _{a}^{b} - \underbrace{\frac{1}{i \lambda} \int_{a}^{b} \left( \frac{A}{s'} \right)' e^{i \lambda s}dt}_{O\left( \frac{1}{\lambda^2} \right)}$$

б) $\exists t_{0}! \in [a,b]: s'(t_{0})=0$

![[Снимок экрана 2026-02-02 в 20.46.12.png|400]]
$$F(\lambda)=A_{0} e^{i \lambda s_{0}}\sqrt{ \frac{2\pi}{\lambda |s_{0}''|} } \exp \left[ i \frac{\pi}{4} \text{sign}(s_{0}'') \right]+ O \left( \frac{1}{\lambda^{3/2}} \right)$$
***Замечание:*** Если $s'(t_{0})=0 \text{ и } s''(t_{0})=0,$ тогда: 

$$s(t) \approx s_{0} + \frac{s_{0}^{(n)}(t-t_{0})^n}{n!}$$
$\implies$ сводим к $\displaystyle \int_{-\infty}^{\infty}e^{i z^n}dz$, либо после замены $(t-t_{0})^n= \xi$ можно свести к Гамма функции.

## 1.4 Метод перевала:

***Лекция-2 [09.02.2026]***

$$F(\lambda)= \int_{\gamma}= A(z) e^{\lambda s(z)} dz , \, \lambda\to + \infty;$$Где $\gamma \in \mathbb{C} ; \,A(z) ,s(z) , z \in \mathbb{C}$.

Идея: основной вклад дают точки $z_{0}$, где $s'(z_{0}) = 0.$

Деформируем $\gamma$ так, чтобы основной вклад набирался в окрестности $z_{0}$ 
>[!dff] Допустимый контур
> Контур $\gamma'$ называется допустимым, если 
>  $$\int_{\gamma' } A e^{ \lambda s } dz = \int _{ \gamma } A e^{\lambda s } dz $$
>   $\implies$ при деформации не пересекаем полюсы $A e^{\lambda s }$, концы контура закреплены (либо не выходят из секторов сходимости). 

>[!dff] Минимальный контур
> Контур $\Gamma$ - минимальный, если ${} \underset{\gamma' }{min} ( \underset{z \in \gamma ' } {max}| e^{\lambda s}|) {}$  достигается на  $\Gamma$. 

### 1. Поведение в окрестности стационарной точки:

Пусть $s'(z_{0})=0$, тогда $\displaystyle s (z)\approx s_{0} + s_{0}'' \frac{(z-z_{0})^2}{2}$

Замена $\displaystyle z= z_{0}  + \eta \sqrt{ \frac{2}{s_{0}''} }, \, \eta= x + i y$:

$$ s  \approx s_{0} + \eta^2 = s_{0} + (x^2 - y^2 + 2 i x y)$$
>[!dff] Линии наискорейшего спуска
> Линии вдоль $\nabla Re(s)$. 

>[!dff] Линии Стокса 
>Линии $Im (s )= const$. 

***Замечание:*** в окрестности точек перевала линии Стокса совпадают с минимальным контуром $\Gamma$.

### 2. Вклад точки перевала:

Пусть $s'(z_{0})= 0 \implies s \approx s_{0} + s_{0} '' \displaystyle \frac{(z-z_{0})^2}{2}$. 

$$F(\lambda) \approx \int_{ \Gamma} A(z_{0})e^{ \lambda s_{0}} e^{\lambda s_{0} '' (z-z_{0})^2 /2 }dz $$

Замена $z-z_{0} = \pm \displaystyle \sqrt{ \frac{2}{\lambda} } \eta \sqrt{  \frac{1}{|s_{0} '' | e^{ i \varphi}} }, \, \varphi = arg (s_{0}'' )$

$$F(\lambda)= \pm \sqrt{ \frac{2 \pi }{\lambda s_{0}'' } }A_{0} e^{ \lambda s_{0}}$$
, где знак выбирается по направлению $\Gamma.$

Алгоритм: 

a) Линии Стокса, минимальный контур; 

б) Разложение $s$ в точке перевала;

***Замечание 1:*** Вклады точек перевала суммируются.

***Замечание 2:*** Вклад от концов контура находится интегрирование по частям.

***Замечание 3:*** $s'(z_{0}) = 0 , \, s''(z_{0})=0 ,\dots$

 $s(z ) \approx s_{0} + \displaystyle s_{0}^{(n)} \frac{(z-z_{0})^n}{n!}$

Замена $(z-z_{0})^n = \eta$, сводится к $\Gamma$-функции. 

# 2. Метод усреднения

$$\frac{d}{dt  } \boldsymbol{ x } = \boldsymbol{ u} ( \boldsymbol{ x} ) + \varepsilon \boldsymbol{ v} , \quad \boldsymbol{ x} = \left\{ x_{1}, \dots, x_{m} \right\} , \, \varepsilon \ll 1 $$

Решение при $\varepsilon = 0$ известно. 

Нужно решение при $\displaystyle t \leq \frac{1}{\varepsilon}$.

"Наивный подход": $\boldsymbol{ x } ( t) =\boldsymbol{ x} _{ 0 } (t ) + \varepsilon \boldsymbol{ x} _{1 } (t ) + \varepsilon ^2 \boldsymbol{ x} _{2 } (t )$ . 

$\displaystyle \frac{ d \boldsymbol{ x }_{0} }{t } = \boldsymbol{ u } ( \boldsymbol{ x } _{0}), \, \boldsymbol{ x}_{0 } (t )$ - известно. 

$$\frac{ d \boldsymbol{ x } _{0 }}{dt } + \varepsilon \frac{ d \boldsymbol{ x } _{1 } }{ dt }  + \dots = \boldsymbol{ u} ( \boldsymbol{x }_{0}) + \varepsilon \frac{ \partial \boldsymbol{u} }{ \partial \boldsymbol{ x}  } \boldsymbol{ x }_{1 } + \varepsilon \boldsymbol{ v } (\boldsymbol{ x}_{0}) + \dots $$

$\varepsilon^1 : \, \displaystyle \frac{ d \boldsymbol{ x } _{1 }}{dt } = \frac{ \partial \boldsymbol{ u} }{ \partial \boldsymbol{x} } \boldsymbol{ x} _{1 } + \boldsymbol{ v } (\boldsymbol{x} _{0 })$ - линейно по $\boldsymbol{ x } _{1}$.

$\varepsilon ^2 : \dots$

Не работает почти никогда.

## 2.1 Стробоскопическое уравнение: 

$$\frac{d \boldsymbol{ x } }{dt } = \boldsymbol{ u} ( \boldsymbol{ x } ) + \varepsilon \boldsymbol{ v } ( \boldsymbol{x }), \quad \boldsymbol{ x  } = \left\{ x_{1}, \dots , x_{n} \right\}. $$

Перейдем к переменной действие-угол: 

$$\begin{cases}
\displaystyle \frac{d \boldsymbol{ I }}{dt } =\varepsilon g ( \boldsymbol{I } , \boldsymbol{ \varphi} )  \\
 \displaystyle \frac{ d \boldsymbol{ \varphi } }{dt } = \boldsymbol{ \omega} ( \boldsymbol{ I} )  +\varepsilon f( \boldsymbol{ I} , \boldsymbol{\varphi} ) 
\end{cases} \quad \quad  \boldsymbol{I} = \left\{ I_{1}  , \dots , I_{m/ 2 }\right\} ; \, \boldsymbol{\varphi} = \left\{ \varphi_{1} ,\dots,\varphi_{m /2} \right\} $$

$$а) \,m =2  \quad \begin{cases}
\displaystyle \frac{dI}{dt } = \varepsilon g ( I , \varphi)  \\
\displaystyle \frac{d\varphi}{dt } = \omega ( I ) + \varepsilon f ( I , \varphi)
\end{cases}$$

$I$ меняется медленно; за время $\displaystyle \frac{ 2 \pi}{\omega}$ $I$ меняется слабо. 

$$\frac{1}{T} \int_{t }^{t+T} \frac{dI}{dt} = \frac{I (t+ T )- I(t)}{t } = \frac{\varepsilon}{t} \int g(I , \varphi) dt = \frac{\varepsilon}{T } \int_{0} ^{ 2 \pi} g( I, \varphi) \left(  \frac{d \varphi }{dt } \right)^{-1} d \varphi$$
$$\frac{1}{T} \int_{t }^{t+T} \frac{dI}{dt} =\varepsilon <g ( I , \varphi)>_{\varphi}$$

б) $m\geq 4$ - несколько частот. 

Если частоты не соизмеримы, то есть нет $m_{i}: \, \sum m_{i} \omega_{i} =0$ (нет резонансов), то 

$$\frac{1}{T } \int_{0}^T g(\boldsymbol{ I} , \boldsymbol{ \varphi} ) dt = <g ( \boldsymbol{ I } ,\boldsymbol{\varphi})> _{\boldsymbol{ \varphi}}$$
$$ \frac{ d \boldsymbol{ I }}{dt } = \varepsilon \boldsymbol{G } ( \boldsymbol{I}) ; \quad \boldsymbol{G} (\boldsymbol{ I})= <\boldsymbol{ g }( \boldsymbol{I} ,\boldsymbol{\varphi})>_{\boldsymbol{\varphi}}$$

## 2.2 Метод Богалюбова-Крылова

Гармонический осциллятор с возмущениями

$$\ddot{x}  +x = \varepsilon F( x, \dot{ x }) $$

$z = z + i \dot{ x }; \quad \dot{ z } = \dot{ x } - i \dot{ x } + i \varepsilon F = - i z  + i \varepsilon F (Re \, z , Im \, z )$

$z = I e^{ i\varphi}$

$z = \dot{ I } e^{ i\varphi } + i \dot{ \varphi} I e^{ i \varphi} = - i I e ^{ i\varphi } + i \varepsilon F ( I \cos \varphi , I \sin \varphi)$

$\dot{ I} + i \dot{ \varphi } = - i I + i \varepsilon e^{  - i\varphi } F$ 

$$\begin{cases}
\displaystyle Re \, : \, \dot{ I} = F (I \cos \varphi , I \sin \varphi  ) \sin \varphi  \\
\displaystyle Im \, : \, \dot{ \varphi }=-1  + \frac{ \varepsilon \cos \varphi}{ I } F( I \cos \varphi , I \sin \varphi) 
\end{cases}$$
Усредненное уравнение: $\displaystyle \frac{dI}{dt } = \varepsilon <F(I \cos \varphi, I \sin \varphi) \sin \varphi >_{\varphi}$ . 

## 2.3 Эффективный потенциал:

Невозмущенная система + высоко частотное возмущение: 

$$\frac{dx}{dt } = u ( x ) + \varepsilon v ( x,t ) ; \quad x( t ) =\underset{медл}{X(t)} + \underset{быстр}{x_{1}(t)} $$

$$\left| \frac{dx_{1}}{dt } \right| \gg \left| \frac{dX}{dt } \right| , \quad |x_{1} | \ll | X|$$
