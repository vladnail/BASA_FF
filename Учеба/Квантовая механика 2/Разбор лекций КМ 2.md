# 1. Спин 
***Лекция-1 [02.02.2026]***

Квантовое описание частицы основанное не на зависимости от импульса и координат, а от внутренней степени свободы. Эта степень свободы получила название ***спина***.

## 1.1 Опыт Штерна-Герлаха:

**Опыт Штерна–Герлаха показал, что магнитный момент атомов не может объясняться только орбитальным угловым моментом.**

Для атома серебра с нулевым орбитальным моментом ожидалось одно пятно, однако экспериментально наблюдалось **два дискретных пятна**, что указывает на существование **дополнительной внутренней степени свободы**.

Эта степень свободы — **спин**, угловой момент, не связанный с координатным движением, проекция которого может принимать **полуцелые значения**; для электрона $\displaystyle s =\frac{1}{2}$.


>[!error] Важно
>Дальше мы будем использовать обезразмеренную величину спина, измеряя его в единицах $\hbar$.

## 1.2 Угловой момент:

>[!dff] Оператор спина
> $$\hat{\mathbf{s}}= (\hat{s}_{x} ,\hat{s}_{y}, \hat{s}_{z})$$

*Суммарный угловой момент:*  $\mathbf{\hat{j}} = \mathbf{\hat{l}} +\mathbf{\hat{s}}$. 

Коммутационное соотношение для компонент оператора $\hat{ \mathbf{j}}$: 

$$[\hat{j}_{i} ,\hat{j}_{j}]= i \varepsilon_{ijk} \hat{j }_{k}$$

Квадрат углового момента: 

$$\mathbf{\hat{j }}^2 = \hat{j}_{x}^2 + \hat{j}_{y}^2 +\hat{j}_{z}^2$$
Откуда: $\displaystyle [\hat{ \mathbf{ j}}^2 , \hat{j }_{i }]= 0  ,\quad i = x, y ,z$

Следовательно у операторов $\mathbf{ \hat{ j} }^2$ и $\hat{j}_{z }$ есть базис собственных значений $\ket{jm }$:

$$\begin{align}
\hat{ \boldsymbol{j}}^2  \ket{ jm }& = j (j +1 ) \ket{ jm }   \\
\hat{j} _{z } \ket{ jm }  &= m \ket{ jm } 
\end{align}$$
___ 
Операторы понижения и повышения: 

$$\hat{j} _{\pm}= \hat{ j} _{x } \pm i \hat{ j} _{y} $$
$$\hat{j } _{\pm} \ket{ jm }  = \sqrt{  (j \mp m) ( j \pm m +1 ) } \ket{ jm \pm 1} $$
Проекция может принимать только следующие значения: 

$$m = - j, j +1 ,\dots , j -1 , j$$
## 1.3 Спин 1/2 и спиноры: 

>[!dff] Базис 
>Общий базис $\hat{\mathbf{ s}}^2$ и $\hat{ s }_{z}$ из двух ортонормированных состояний: 
> $$\ket{s =\frac{1}{2}, s_{z}=+\frac{1}{2}} \equiv \ket{\uparrow} , \quad \ket{s =\frac{1}{2}, s_{z} = - \frac{1}{2}} \equiv \ket{\downarrow}  $$

В дираковских обозначениях имеем: 

$$\hat{ s} _{z } \ket{\uparrow}  = \frac{1}{2 } \ket{ \uparrow}  , \quad \hat{s} _{z} \ket{ \downarrow} = - \frac{1}{2 } \ket{ \downarrow} $$

---
Общее состояние: 

$$\ket{\chi }  = a \ket{\uparrow} + b \ket{ \downarrow}  $$
, где комплексные коэф-ты удовлетворяют нормировке $|a|^2 + |b |^2 =1$. 

>[!dff] Спинор
>Состояние, характеризуемое амплитудами $a$ и $b$, удобно записывать в
виде двухкомпонентного столбца: 
> $$ \chi= \begin{pmatrix}
> a  \\
b
>\end{pmatrix}$$

Заметим: 

$$\braket{ s_{z} | \chi } = \begin{pmatrix}
\braket{ \uparrow | \chi }  \\
\braket{ \downarrow | \chi }  
\end{pmatrix} = \begin{pmatrix}
a  \\
b
\end{pmatrix}$$
---
Общий вид матрицы $\hat{ s }_{i } \, ( i = x, y ,z )$: 

$$\hat{ s } _{ i } = \begin{pmatrix}
\bra{\uparrow} \hat{ s } _{i} \ket{\uparrow}  & \bra{\uparrow} \hat{ s } _{ i} \ket{\downarrow}  \\
\bra{\downarrow}  \hat{s}  _{i } \ket{\uparrow}  &  \bra{\downarrow}  \hat{s}  _{i} \ket{ \downarrow} 
\end{pmatrix}$$

## 1.4 Матрицы Паули: 

>[!dff] Матрицы Паули: 
> $$\hat{ \sigma}_{x} = \begin{pmatrix}
> 0  & 1  \\
> 1  & 0 
> \end{pmatrix}, \quad \hat{\sigma}_{y} = \begin{pmatrix}
> 0  &  - i  \\
> i  & 0
> \end{pmatrix}, \quad \hat{\sigma}_{z}= \begin{pmatrix}
> 1  & 0 \\
> 0 & -1
> \end{pmatrix}$$


Связь со спином: 

$$\hat{\boldsymbol{s}}= \frac{1}{2 } \hat{\boldsymbol{\sigma}}  \quad \left( \text{для } s=\frac{1}{2} \right)$$
Важные  свойства (для матриц $2\times 2$): 

1) $\hat{\sigma}_{i } ^{^{\dagger}} = \hat{\sigma}_{i }, \quad Tr \,\hat{\sigma}_{i} =0 ,\quad \hat{\sigma}_{i}^2 =\hat{I}, \quad i= x,y,z$ ;
2)  $\hat{\sigma}_{i} \hat{\sigma}_{j}= \delta_{ij}\hat{I}+ i \varepsilon_{_{ij k}} \hat{\sigma} _{k}$ ;
3) $[ \hat{\sigma} _{i }, \hat{\sigma}_{j}]= 2 i \varepsilon_{i j k} \hat{\sigma}_{k}$

## 1.5 Нормировка и скалярное произведение спиноров: 

Рассмотрим случай когда спинор зависит только от амплитуд $a$ и $b$:

Нормировка: 

$$\chi^{^{\dagger}} \chi = | a|^2 + |b|^2 = 1$$

Скалярное произведение: 

$$\braket{ \chi_{1} | \chi_{2} } = \chi^{^{\dagger}}_{1 } \chi_{2}$$

--- 
Общий случай с координатной зависимостью: 

Нормировка: 

$$\int d^3 \mathbf{r} \, \chi^{^{\dagger}} (\mathbf{r}) \chi (\mathbf{r}) =1$$

Скалярное произведение: 

$$\braket{ \chi_{1} |  \chi_{2}}= \int d^3 \mathbf{r} \, \chi^{^{\dagger}}_{1} (\mathbf{r}) \chi_{2} (\mathbf{r}) = \int d ^3 \mathbf{r} (a_{1}^* (\mathbf{r}) a_{2}(\mathbf{r})+ b_{1}^* (\mathbf{r}) b_{2 }(\mathbf{r}))$$
## 1.6 Оператор поворота и полный угловой момент: 

Орбитальный поворот (вокруг оси $\boldsymbol{n}$ на угол $\alpha$): 

$$\hat{R}_{orb} (\boldsymbol{\alpha}) = \exp \left[ i \mathbf{\hat{l}} \boldsymbol{\alpha} \right] , \quad \boldsymbol{\alpha} = \alpha \boldsymbol{n}$$

Спиновой поворот (вокруг оси $\boldsymbol{n}$ на угол $\alpha$ в спиновом пространстве): 

$$\hat{R}_{spin} (\boldsymbol{\alpha}) = \exp \left[ i \hat{ \mathbf{s}} \boldsymbol{\alpha} \right] $$

Полный угловой поворот: 

$$\hat{R}_{total } = \exp \left[ i \hat{ \mathbf{j }} \vec{\alpha} \right] $$
## 1.7 Преобразование спиноров при поворотах: 

Общая формула для частиц со спином 1/2: 

$$\begin{align}
\hat{ R } (\alpha \vec{n} )&= \exp \left[ \frac{i}{2 } \alpha \vec{ \sigma} \vec{n } \right]  \\
\hat{ R} (\alpha  \vec{n})&= \hat{ I} \cos \frac{\alpha}{2 } + i (\vec{\sigma} \vec{\cdot}n) \sin \frac{\alpha}{2}
\end{align}$$
>[!error] Важно
>Важное следствие формулы:
> $$\hat{ R } _{spin} (2 \pi \vec{n}) = - \hat{ I}$$

## 1.7.1 Состояние с максимальной проекцией спина на заданную ось: 

В качестве примера найдем явный вид спинора $\chi_{+}(\boldsymbol{\xi})$, описывающий состояние с определенной проекцией спина +1/2 на некоторую ось $\boldsymbol{ \xi}$:

$$\boldsymbol{\xi}= \begin{pmatrix}
\sin \theta \cos \phi \\
\sin \theta \sin \phi  \\
\cos \theta
\end{pmatrix}$$
Чтобы получить итоговый поворот системы координат, так чтобы исходная ось $z$ совпала с направлением $\boldsymbol{\xi}$ , последовательно сделаем повороты: сначала на угол $- \theta$ вокруг оси $y$, а затем на угол $- \phi$ вокруг новой оси $z$: 

$$\begin{align}
\hat{R}( - \theta \boldsymbol{e}_{y}) &= \exp \left( - \frac{i}{2} \hat{ \sigma}_{y} \theta  \right) = \cos \frac{\theta}{2} \hat{ I} - i \hat{ \sigma}_{y} \sin \frac{\theta}{2} = \begin{pmatrix}
\cos \frac{\theta}{2}  &  - \sin \frac{\theta}{2}  \\
\sin \frac{\theta}{2}  &  \cos \frac{\theta}{2}
\end{pmatrix}  \\
\hat{ R} (- \phi \boldsymbol{e}_{z}) & = \exp \left( - \frac{i}{2} \hat{\sigma}_{z} \phi \right)  = \cos \frac{\phi}{2} \hat{I} - i \hat{\sigma}_{z} \sin \frac{\phi}{2} = \begin{pmatrix}
e^{- i \phi /2 }  & 0 \\
0  & e^{i \phi /2 }
\end{pmatrix}
\end{align}$$

Последовательно действуем: 

$$\chi _{+} (\boldsymbol{\xi}) = \hat{ R}( - \phi \boldsymbol{e}_{z}) \hat{ R} ( - \theta \boldsymbol{e}_{y}) \begin{pmatrix}
1 \\
0 
\end{pmatrix} = \begin{pmatrix}
e^{ - i \phi /2  } \cos \frac{\theta}{2} \\
e^{i \phi /2  } \sin \frac{\theta}{2}
\end{pmatrix}$$

--- 
Можно так же получить этот ответ если рассматривать состояние $\chi _{+} (\boldsymbol{\xi})$ как собственное состояние оператора проекции спина на ось $\boldsymbol{\xi}$:

$$(\hat{\boldsymbol{s} } \cdot \boldsymbol{ \xi}) \chi_{+} ( \boldsymbol{ \xi})  = \frac{1}{2} \chi_{+} ( \boldsymbol{\xi}) = \frac{1}{2 } ( \hat{ \boldsymbol{\sigma }} \cdot \boldsymbol{ \xi}) \chi _{+} ( \boldsymbol{\xi})$$

Матрица оператора имеет вида: 

$$\hat{ \boldsymbol{ \sigma } } \cdot \boldsymbol{\xi}= \begin{pmatrix}
\cos \theta  &  e^{- i \phi } \sin \theta  \\
e^{ i \phi } \sin \theta  &  - \cos \theta
\end{pmatrix}$$
Подставляем спинор общего вида и учитываем $|a|^2 + |b|^2 =1$ получаем тот же самый ответ.

---

***Общий вид спинора для $s =\displaystyle \frac{1}{2}$:***

Любой нормированный спинор частицы со спином $1 /2$: 

$$\chi = \begin{pmatrix}
a \\
b 
\end{pmatrix}, \quad | a|^2 + | b|^2 =1$$
можно (с точностью до общего фазового множителя) представить в виде:

$$\chi = \begin{pmatrix}
e^{-i \phi/2}\cos \frac{\theta}{2} \\
e^{i \phi/2 } \sin \frac{\theta}{2}
\end{pmatrix}, \quad 0\leq \theta\leq \pi, \, 0 \leq \phi \leq 2 \pi $$

Следовательно, любой чистый спиновый $\displaystyle s =\frac{1}{2}$ - состояние полностью поляризовано вдоль некоторого направления в пространстве.


## 1.8 Проекционные операторы для спина 1/2: 

>[!dff] Проектор 
> Основное свойство проектора $\hat{P}$ состоит в том, что его двухкратное применение к любому состоянию даст тот же результат, что и однократное $\hat{ P} ^2 = \hat{ P }$. 
> 
>Физически это означает, что если состояние уже спроектировано на некоторое подпространство, то повторная проекция на то же подпространство его не изменяет.

Разложение проектора по $\hat{ I}$ и $\hat{ \boldsymbol{ \sigma}}$:

$$\hat{P} _{\pm} (\vec{\xi})= \frac{1}{2  } \left( \hat{I} \pm \vec{\xi} \cdot\hat{\vec{\sigma}} \right) $$

, где $\boldsymbol{\xi}$ - некоторый единичный вектор.

Свойства операторов: 

1) $\hat{ P} ^2 _{\pm} = \hat{P}_{\pm}$ ; 
2) $\hat{P}_{+} \hat{P} _{-}= 0$ ;
3) $\hat{P} _{+} + \hat{P}_{-}= \hat{I}$ ; 
---

**Физический смысл проекционных операторов.**  
Оператор проекции спина на направление $\boldsymbol{\xi}$:
$$
\hat s_\xi = \frac12(\boldsymbol{\xi}\cdot\hat{\boldsymbol{\sigma}})
$$
Проекционные операторы $\hat P_\pm(\boldsymbol{\xi})$ выделяют из произвольного спинора $\chi$
собственные состояния оператора $\hat s_\xi$:
$$
\hat s_\xi\big(\hat P_\pm(\boldsymbol{\xi})\chi\big)
= \pm\frac12\,\hat P_\pm(\boldsymbol{\xi})\chi.
$$

$\hat{P}_{+}(\boldsymbol{\xi})$ дает вектор с направлением $\boldsymbol{\xi}$ равный по модулю проекции состоянию +1/2 на ось $\boldsymbol{ \xi}$, $\hat{P}_{-}(\boldsymbol{\xi})$ аналогично.

## 2.1 Калибровочная инвариантность и уравнение Шрёдингера:
