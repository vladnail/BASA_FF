# 3. Решение интегралов методами математического анализа (ОМА)
## 3.1 Интегрирование рациональных функций:

***1. Метод неопределенных коэффициентов:***

$\displaystyle \int \frac{P(x)}{Q(x)} dx$  - здесь $P(x)$ и $Q(x)$ многочлены с вещественными коэффициентами, причем степень многочлена $P(x)$ меньше степени многочлена $Q(x)$.
Последовательность действий:

$$Q(x ) = a (x-\alpha)^n  ...(x^2 + px +q )^m,$$
где $a , \, \alpha , \, p , \, q \in \mathbb{R}$.
$$\begin{align}
\frac{P(x)}{Q(x)}&=\frac{a_{1}}{x-\alpha}+ \frac{a_{2}}{(x-\alpha)^2}+\dots+\frac{a_{n}}{(x-\alpha)^n}+\dots +  \\
&+ \frac{b_{1} x + c_{1}}{x^2 + px +q} + \frac{b_{2} x +c_{2} }{(x^2 + px + q)^2}+ \dots+ \frac{b_{m }x + c_{m}}{(x^2 + px +q )^m}
\end{align}$$
 Приравниваем коэффициенты при одинаковых степенях $x$, находим константы. 
 
В силу линейности интеграла получим линейную комбинацию следующих интегралов:
$$\begin{gathered}
\int \frac{dx}{x-\alpha} = \ln | x-\alpha| \\ 
\int \frac{dx}{(x-\alpha)^j}= \frac{(x-\alpha)^{1-j}}{1-j} \\ 
\int \frac{bx +c }{x^2 + px + q } - \text{интеграл вычисляется с помощью } \\
\text{выделения полного квадрата в квадратном трехчлене.}
\end{gathered}$$

***2. Метод Остроградского:***

Метод целесообразно использовать в случае, когда знаменатель $Q(x)$ имеет кратные корни, особенно комплекно-сопряженные. Представим интеграл в виде: 
$$ \tag{!}\int \frac{P(x)}{Q(x)}dx = \frac{P_{2}(x)}{Q_{2}(x ) } + \int \frac{P_{1}(x)}{Q_{1}(x)}$$
, где многочлен $Q_{1} (x )$ имеет все те же корни, что и многочлен $Q(x)$, но в первой степени, $\displaystyle Q_{2}(x ) = \frac{Q(x)}{Q_{1}(x)}$, многочлены $P_{1}(x) ,\, P_{2}(x)$ имеют степень на 1 меньше, чем многочлены $Q_{1}(x) , \, Q_{2} (x)$ соответственно. 

Таким образом суть метода Остроградского в выделении алгебраической части исходного интеграла. 

На практике удобно дробь $\displaystyle \frac{P_{1}(x)}{Q_{1}(x)}$ сразу разложить на элементарные слагаемые: 
$$\frac{P_{1}(x)}{Q_{1}(x) }dx =   \frac{a_{1}}{x-a_{1}}+\dots+\frac{a_{n}}{x-a_{n}}+ \frac{b_{1} x +c_{1}}{x^2 + p_{1} x + q_{1} } +\dots \frac{b_{m} +c_{m}}{x^2 + p_{m} x + q_{m} } $$
После чего найти коэффициенты многочлена $P_{2}(x)$ и коэффициенты $a_{j }, \, b_{k}, \, c_{k}$, дифференцируя полученное равенство после подстановки разложения $\displaystyle \frac{P_{1}(x)}{Q_{1}(x)}$ в $(!)$.

## 3.2 Интегрирование иррациональных функций:

1. $\displaystyle \int \frac{(px+q)dx}{\sqrt{ ax ^2 +b x +c  }}, \, \int \sqrt{ a x^2 + bx + c } \,dx$ - выделение полного квадрата
2. $\displaystyle\int \mathrm{R}\left( x, \left( \frac{ax +b }{cx +d } \right)^{\frac{n_{1}}{m_{1}} }, \left( \frac{ax +b }{cx +d } ^{\frac{n_{2}}{m_{2}}}\right),\dots\right)dx$ - сведение к рациональной функции подстановкой $\displaystyle t^p = \frac{ax +b }{cx +d }$, где $p$ - наименьшее общее кратное целых чисел $m_{1}, \, m_{2} ,\dots$
3. $\displaystyle \int\frac{dx}{(x-a)^n \sqrt{ a x^2 + bx +c  }}$ - подстановка $\displaystyle t =\frac{1}{x-a}$
4. Сведение к интегрированию тригнометрических функций:
 $\displaystyle \int \mathrm{R} ( x, \sqrt{ a^2 -x^2 })dx$ - подстановка $x = a \sin t$; 
$\displaystyle\int \mathrm{R}(x, \sqrt{  x^2 + a^2 })dx$ - подстановка $x =a \tan t$; 
$\displaystyle \int \mathrm{R}(x, \sqrt{  x^2 - a ^2 }) dx$ - подстановка $\displaystyle x =\frac{a}{\cos t}$.
5. $\displaystyle \int \mathrm{R}(x, \sqrt{  ax ^2 + bx +c  }) dx$ - сведение к рациональной функции с помощью подстановок Эйлера: 
$$\sqrt{  a x ^2 + b x +c  } = 
\begin{cases}
\pm \sqrt{  a }x + z  & ,a>0,  \\
xz \pm \sqrt{  c  }  &  ,c>0;
\end{cases}$$
$$ \sqrt{  a ( x -x_{1} )( x -x_{2}) } = z ( x -x_{1})$$
6. Интеграл от дифференциального бинома $\displaystyle \int x^m (a +b x^n )^p dx$, $a \neq 0 , \, b \neq 0 , \, n \neq 0 , \, p \neq 0$ ; $m , \, n, \, p$ - рациональные числа; $a, \, b$ - действительные числа. 
	1.  Если $p$ - целое, то $x= z^{N}$ , $N$ - общий знаменатель дробей $m$ и $n$ 
	2.  Если $\displaystyle \frac{m+1}{n}$ - целое, то $a +b x^n=z^N$, $N$ - знаменатель дроби p
	3. Если $\displaystyle \frac{m+1}{n}+p$ - целое, то $\displaystyle \frac{a}{x^n}+b =z^N$, $N$ - знаменатель дроби p.
Заметим, что если $n=1$, то следует проверить случаи: 1) $p$ - целое; 2) $m$ - целое; 3) $m+p$ - целое.

## 3.3 Интегрирование тригонометрических фунций: $\displaystyle \int \mathrm{R} (\sin x, \cos x)dx$
 ***1. Универсальная тригонометрическая подстановка:***
$$\begin{align}
&\tan \frac{x}{2 }= t \implies x = 2 \arctan t \implies dx= \frac{2dt}{1 +t ^2}  \\
& \sin x = \frac{2 \sin \frac{x}{2} \cos \frac{x}{2}}{\sin^2 \frac{x}{2}+\cos^2 \frac{x}{2}} = \frac{2t}{1+t ^2} \\
& \cos x = \frac{\cos ^2 \frac{x}{2} - \sin ^2 \frac{x}{2 }}{\sin ^2 \frac{x}{2} + \cos^2 \frac{x}{2}} = \frac{1 -t^2}{1+t ^2} \\
&\tan x = \frac{2t}{1 -t ^2}
\end{align}$$
***2. В следующих случаях удобны следующие подстановки:***

а) если $\mathrm{R} (- \sin x , \cos x) = - \mathrm{R}(\sin x ,\cos x )$, то $\cos x =t$;

б) если $\mathrm{R}(\sin x , -\cos x )= - \mathrm{R}(\sin x ,\cos x )$, то $\sin x = t$;

в) если $\mathrm{R}(- \sin x , - \cos x ) = \mathrm{R}(\sin x ,\cos x)$, то: 
$$\begin{align}
&\tan x =t \implies x = \arctan t \implies dx = \frac{dt}{1 +t ^2} \\
& \sin^ 2 x = \frac{\sin^2 x}{ \sin^2x + \cos ^2 x } = \frac{t^2}{1 +t ^2} \\
& \cos ^2x =\frac{\cos^2 x}{ \sin^2x + \cos ^2 x }= \frac{1}{1+t ^2}
\end{align}$$
***3. Понижение степени:***

Если интеграл имеет вид $\displaystyle \int \sin^m x\cos ^n x dx$, то используем формулы для понижения степени:
$$\cos ^2 x = \frac{ 1 + \cos 2x }{2} , \quad \sin^2 x = \frac{1 -\cos ^2 2x }{2} , \quad 2 \sin x \cos x = \sin 2 x $$
Дополнительная полезная формула:

$$\int_{0}^{\pi/2} \sin^m x \cos ^n x dx = \text{B} \left( \frac{m+1 }{2}, \frac{n+1}{2} \right) $$

# 4. Решение интегралов методами теории функций комплексной переменной (ТФКП)

## 4.2 Вычеты

 >[!example] Виды и.о.т.
 > Изолированная особая точка $z_{0}$ функции $f(z)$ называется:
 > 1) _устранимой особой точкой_, если $\displaystyle \lim_{ z \to z_{0} } f(z)$ существует и конечен;
 > 2) _полюсом_, если $\displaystyle\lim_{ z \to z_{0} }f(z)= \infty$;
 > 3) _существенно особенной точкой_, если функция $f(z)$ не имеет предела в точке $z_{0}$.



>[!theorem] Вычет в простом полюсе
> $$
> \displaystyle\underset{z=z_0}{Res} f(z)
> =
> \lim_{z\to z_0}(z-z_0)f(z).
> $$

**Частный случай.**  
Если
$$f(z)=\frac{\varphi(z)}{\psi(z)}, \qquad
\psi(z_0)=0,\ \psi'(z_0)\neq 0,$$
,то
$$\operatorname{Res}_{z=z_0} f(z)
=
\frac{\varphi(z_0)}{\psi'(z_0)}.$$

>[!theorem] Вычет в полюсе порядка \(n\)
> $$
> \displaystyle\underset{z=z_0}{Res}f(z)
> =
> \frac{1}{(n-1)!}
> \lim_{z\to z_0}
> \frac{d^{\,n-1}}{dz^{\,n-1}}
> \Bigl[(z-z_0)^n f(z)\Bigr].
> $$


---

***1. Связь вычета с типом особой точки:***


>[!note] Устранимая особая точка
> 1) Если точка $z_0\neq\infty$ является устранимой особой точкой функции $f(z)$,
> то
> $$
> \underset{z=z_0}{\mathrm{Re}s} f(z)=0.
> $$
> 2) Если точка $z_0=\infty$ является устранимой особой точкой функции $f(z)$,
> то
> $$ z_{0} = \infty \implies \underset{z_{0}}{Res} \neq 0 $$
> Тогда: 
> $$\underset{\infty} {Res} f(z)= \lim_{ z \to \infty } \left( z (f(\infty)- f(z)) \right) $$


***2. Важные теоремы:***

>[!abstract] Теорема о полной сумме вычетов
> $$\sum_{k=1}^{n} \underset{z_{k}}{Res}f(z) + \underset{\infty}{Res}f(z)=0$$

>[!abstract] Основная теорема о вычетах
> ![[Снимок экрана 2026-01-28 в 23.26.52.png]]  
> $$\int_{ \Gamma} f(z )dz = 2 \pi i \sum_{k=1}^n \underset{z_{k}}{Res}f(z)$$

$$\begin{align}
\int_{0}^{2 \pi} \mathrm{R}(\sin t , \cos t) dt &=\frac{1}{i} \int_{|z|=1}\underbrace{ \frac{1}{z}\mathrm{R} \left( \frac{z- 1/z}{2 i}, \frac{z + 1/z}{2} \right) }_{R_{1}(z)}dz=  \\
