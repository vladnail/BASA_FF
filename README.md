# БАЗА ДАННЫХ (Obsidian Vault)

Личный справочник/база знаний по физике и матану в формате Obsidian: формулы, определения, теоремы, шпаргалки, примеры решения и структурированные конспекты.

## Что внутри

- `БАЗА/` — основная база (справочники, формулы, практические заметки)
- `Учеба/` — материалы по курсам/семестрам
- `Литература/` — источники, выдержки, ссылки
- `3 курс 1 семестр/` — текущие конспекты/задачи (если используется)

> Репозиторий предполагает работу через Obsidian (Markdown + LaTeX).
